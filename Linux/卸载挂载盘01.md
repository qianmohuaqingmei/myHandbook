## 卸载挂载盘

### 1 查看挂载点

```bash
# df -h
```

### 2 卸载挂载点

```bash
# umount /opt/duzi 
```

### 3 查看逻辑卷

```bash
# lvscan
```
### 4 卸载逻辑卷

```bash
# lvremove /dev/vg01/opt_duzi
```

### 5 查看物理卷

```bash
# pvscan
```

### 6 卸载物理卷

```bash
# vgremove vg01
```

### 7 查看分区

```bash
# pvdisplay
```

### 8 删除分区

```bash
# pvremove /dev/vdb1
```

### 9 清除fstab的配置

```bash
# vim /etc/fstab
```
